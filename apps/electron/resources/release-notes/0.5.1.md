# v0.5.1 — Stability Fixes

---

## Improvements

- **Skill metadata** — `requiredSources` and `icon` fields are now displayed in the skill info page and accepted by validation ([#290](https://github.com/lukilabs/craft-agents-oss/issues/290))

## Bug Fixes

- **Install scripts** — Fetch YAML manifests directly instead of relying on the version endpoint, fixing installations after infrastructure changes. PowerShell fixes: multiline regex for version extraction, download YAML to file for PS 5.1 compatibility, wrap script in scriptblock so `exit` doesn't kill the terminal ([#273](https://github.com/lukilabs/craft-agents-oss/issues/273))
- **Windows runtime** — Upgraded vendored Bun 1.3.5 → 1.3.9 to fix intermittent ACCESS_VIOLATION crash (60% crash rate → 0%)
- **Windows build** — Fixed native module resolution for koffi, copy interceptor-request-utils for packaged builds, correct native binary paths for both inlined and external imports
- **Pi SDK 0.55.0** — Fixes streaming freezes during large operations and malformed trailing tool-call JSON in OpenAI partial chunks
