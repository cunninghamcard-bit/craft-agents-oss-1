# 判断规则

## 业务结论

业务结论优先收敛到两类：

- `不影响使用`：公开资料能证明两个型号是同一订货项、同一标准号/商业料号映射、官方替代，或只差写法/包装且不改变实物。
- `影响使用`：公开资料能证明两个型号的关键属性冲突；或者查完后仍不能证明报价型号覆盖采购型号，当前不能按采购型号接受。

不要把资料不足写成第三种业务结论。资料不足要写成不足类型，说明缺的是哪一类资料或使用要求。

## 不足类型

- `无`：资料已经足够支撑 `不影响使用` 或 `影响使用`。
- `差异资料不足`：公开资料没有解释两个型号的差异字段、后缀、字符、订货码或规格差异。问题不是确认型号真假，而是不知道差异到底代表什么。
- `关系证据不足`：差异含义基本清楚，但没有资料证明它们是同一料、替代料、等价写法、cross-reference 或同一订货项。问题是知道差异是什么，但不知道报价型号能不能覆盖采购型号。
- `使用条件不足`：已经找到差异，但不知道客户/工程是否接受这个差异。问题不是公开资料不够，而是缺客户使用要求。

## 门槛

- 有官方映射、官方替代、同订货项、官方 cross-reference，才输出 `不影响使用`。
- 有官方资料或规格资料证明关键属性冲突，才输出 `影响使用`。
- 查完规定资料后，仍不能证明报价型号覆盖采购型号时，业务结论输出 `影响使用`，不足类型按缺口选择 `差异资料不足`、`关系证据不足` 或 `使用条件不足`。
- 只有主体差异、相似字符串、报价型号资料、分销商页面，都不能直接输出 `不影响使用`。
- 后续处理建议只能作为建议，不能写成第三种业务结论。

## 特殊差异

- 后缀差异：如果原厂页写明 replacement 或对应关系，这是关系资料；找不到时不能靠后缀相近接收。
- 写法差异：去横杠、空格、大小写归一化只能生成搜索线索，不能直接作为结论。需要公开资料支持两种写法指向同一实物。
- 数值写法差异：只有四个条件同时满足，才可以输出 `不影响使用`：主体型号相同、单位相同、数值数学等价、公开资料能证明这是同一产品的同一参数字段。缺任一条件时，输出 `影响使用`，按缺口写 `差异资料不足` 或 `关系证据不足`。
- alternate-name / 别名线索：普通平台、库存商、非授权页面写了 alternate name 不能直接放行。必须是原厂或授权渠道把两种写法绑定到同一内部号、同一订货项、官方替代或 cross-reference，才可以输出 `不影响使用`。
- 字符混淆：先看差异含义是否查清。差异字段含义没查清时，业务结论输出 `影响使用`，不足类型写 `差异资料不足`；差异含义清楚但缺同料、替代、同订货项或 cross-reference 证据时，写 `关系证据不足`。不能只靠肉眼相似放行。
- 老备件/弱资料：第三方库存页只能证明线索。人工确认点要要求原厂资料、订货编码表、图纸、铭牌照片、授权渠道确认或供应商书面说明。

## 下一步动作

- `差异资料不足`：找供应商或原厂补 datasheet、订货编码说明、后缀/字符含义说明、规格书、包装标签照片。
- `关系证据不足`：找供应商或原厂补替代证明、厂家书面确认、ordering guide、cross-reference。
- `使用条件不足`：找客户或工程确认这个差异是不是关键要求。

## 置信度

- 高：原厂或官方资料直接证明等价、替代、同一订货项，或直接证明关键差异。
- 中：原厂资料分别证明双方关键规格，差异影响可以由规格对比得出，但没有直接替代/不替代声明。
- 低：只找到分销商、平台、供应商页面，或资料只能说明单边规格，不能解释差异和替代关系。
